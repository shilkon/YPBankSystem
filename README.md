# YPBankSystem

Библиотека для парсинга/сериализации/десериализации финансовых данных в несколько форматов и отдельные консольные приложения, использующие функциональность для конвертации и сравнения файлов.

## Поддерживаемые форматы

Поддержка форматов реализуется через трейты `TransactionReader` и `TransactionWriter`. В библиотеке реализована поддержка форматов `CSV`, `TXT` и `BIN`, спецификации форматов описаны в соответствующих файлах в папке `specifications`.

## Крейт parser

Библиотека, обеспечивающая потоковую сериализацию и десериализацию данных о банковских операциях (транзакциях). Предоставляет реализации для поддерживаемых форматов, не зависящую от конкретного типа источника данных за счёт использования стандартных трейтов ввода/вывода (BufRead/Write).

## Крейт converter

Консольное приложение, использующее библиотечную функциональность для конвертации файла с транзакциями из одного поддерживаемого формата в другой.

## Крейт comparer

Консольное приложение, использую библиотечную функциональность для сравнения данных о транзакциях в двух файлах.
